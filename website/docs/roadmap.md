# 路线图与支持边界

## v0.1 已交付

能力 manifest、认证会话、RBAC、组织与角色治理、菜单树、资源 schema、审计、文件存储、敏感数据保护、生命周期、OpenAPI 生成和 Pages 文档站属于当前底座。

## 当前默认边界

默认发行版是单数据源、单原生事务边界。多数据源租户路由、读写分离、分片、联邦查询、XA、MQ 和搜索投影仍属于后续独立节点，不能从 README 或能力发现中误读为已认证能力。

## 后续优先级

1. 服务合同与 Query/Command Object 的公共化；
2. 数据源 placement、读写路由和迁移安全门；
3. 数据库认证矩阵与可选集成端口；
4. outbox、MQ 和异步搜索投影。
