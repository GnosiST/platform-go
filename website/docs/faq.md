# 常见问题

## 这是一个业务系统吗？

不是。它是业务中立底座，业务资源和流程应通过能力合同接入。

## 默认支持哪些数据库？

默认支持边界以仓库中的认证矩阵和生产 preflight 为准。开发/测试可使用 SQLite；不要把驱动能编译等同于生产认证完成。

## 为什么页面没有直接暴露 SQL？

高风险查询使用服务端持久化 Query Object，客户端提交 query ID、版本和类型化参数，避免把物理 schema、Join 和运算符暴露为公共合同。

## 可以直接接入 MQ 或 ES 吗？

平台保留禁用默认的 ports 和能力合同。接入前需要 adapter、健康状态、重试/限流、审计和故障恢复证据。
