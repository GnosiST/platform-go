# platform-go public site

This directory is a Docusaurus site for the public documentation and GitHub
Pages build. It is intentionally separate from the Gin API and Admin runtime.

```bash
npm install
npm run start
npm run build
```

The initial scaffold provides Chinese and English locale configuration and
portable entry pages. Landing-page art direction, project-owned screenshots,
metadata verification and browser acceptance are release gates for the
`public-docs-site` task and must be completed before publication.
